## Getting Started

1. For production environments - the most stable version
  - Clone repository from `main` branch `git clone https://github.com/dzl-dev/repository-template/tree/main`

2. For staging environments - further testing of confirmed changes from testing stage
  - Clone repository from `staging` branch `git clone https://github.com/dzl-dev/repository-template/tree/staging`

3. For testing environments - for testing upcoming changes not ready for staging
  - Clone repository from `testing` branch `git clone https://github.com/dzl-dev/repository-template/tree/testing`
